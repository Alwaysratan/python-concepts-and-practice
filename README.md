# 🐍 Python Programming - From Basics to Advanced

Welcome to my **Python Learning Repository**! 🚀

This repository contains my structured Python learning journey, starting from the fundamentals and gradually covering advanced concepts. Each topic is organized into a separate Jupyter Notebook with explanations, examples, interview questions, and practice exercises.

---

## 📚 Topics Covered

| No. | Notebook | Description |
|-----|----------|-------------|
| 01 | Variables & Input | Variables, naming rules, multiple assignment, swapping variables, user input |
| 02 | Data Types & Type Casting | Numeric data types, strings, booleans, collections, type conversion |
| 03 | Operators | Arithmetic, comparison, logical, assignment, identity, membership, bitwise operators |
| 04 | Conditional Statements | if, if-else, elif, nested if |
| 05 | Loops | while loop, for loop, range(), break, continue, pass |
| 06 | Strings | String operations, indexing, slicing, string methods |
| 07 | Lists | List operations and built-in methods |
| 08 | Tuples | Tuple operations and methods |
| 09 | Sets | Set operations, methods, mathematical set operations |
| 10 | Dictionaries | Dictionary operations and methods |
| 11 | Functions | User-defined functions, parameters, return statements, lambda |
| 12 | Modules & Packages | Importing modules, creating modules |
| 13 | File Handling | Reading and writing files |
| 14 | Exception Handling | try, except, finally, raise |
| 15 | Object-Oriented Programming | Classes, objects, inheritance, polymorphism |
| 16 | Mini Projects | Python practice projects |
